# jp-webscript-profile-page
A clean, professional personal profile page showcasing skills, projects, and contact information, built using semantic HTML5, CSS3, and HTML5 form validation.

## Features
*   **Semantic HTML**: Uses `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, and `<footer>` for better accessibility and SEO.
*   **Responsive Design**: Simple layout that adapts to different screen sizes.
*   **Functional Form**: Includes "Contact Me" section with Name, Email, and Message fields using HTML5 validation (`required`, `minlength`, `type="email"`).
*   **Image Optimization**: Includes a Profile Picture with proper `alt` text.

## Included Sections
*   Header (Title, Intro, Navigation)
*   About Me
*   Skills (Unordered List)
*   Projects (Articles)
*   Contact Form
*   Footer (Copyright & Social Link)
